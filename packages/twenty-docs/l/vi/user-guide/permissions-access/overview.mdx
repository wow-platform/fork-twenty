## Tính năng chính

- **Truy cập dựa trên vai trò**: Tạo vai trò tùy chỉnh với quyền cụ thể
- **Quyền đối tượng**: Kiểm soát ai có thể xem, chỉnh sửa hoặc xóa bản ghi
- **Quyền trường**: Giới hạn truy cập vào các trường nhạy cảm
- **Quyền cài đặt**: Kiểm soát truy cập vào cấu hình workspace
- **Tích hợp SSO**: Cấu hình đăng nhập một lần cho bảo mật doanh nghiệp (gói Organization)
