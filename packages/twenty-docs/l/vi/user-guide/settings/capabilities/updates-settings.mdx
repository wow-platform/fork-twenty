## Giới thiệu về cài đặt cập nhật

Phần Cập nhật cho phép bạn bật tính năng Truy cập sớm trước khi chúng được phát hành chính thức.

## Tính năng Truy cập sớm

Tính năng Truy cập sớm là các khả năng đang được thử nghiệm trước khi phát hành chính thức.

**Hiện đang ở giai đoạn Truy cập sớm:**
- **Dashboard** — Báo cáo và phân tích trực quan

### Cách bật tính năng Truy cập sớm
1. Đi tới **Cài đặt → Cập nhật**
2. Điều hướng đến phần **Truy cập sớm**
3. Tìm tính năng bạn muốn bật
4. Bật công tắc
5. Tính năng sẽ khả dụng ngay lập tức

<Note>
Tính năng Truy cập sớm có thể thay đổi dựa trên phản hồi người dùng. Phản hồi của bạn giúp chúng tôi cải thiện Twenty!
</Note>

## Phản hồi tính năng

Phản hồi của bạn giúp cải thiện Twenty:
- Báo cáo vấn đề với tính năng Truy cập sớm
- Chia sẻ cách bạn sử dụng tính năng mới
- Đề xuất cải tiến qua cộng đồng Discord
