Sử dụng trình soạn thảo văn bản phong phú dựa trên block từ [BlockNote](https://www.blocknotejs.org/) cho phép người dùng chỉnh sửa và xem các khối nội dung.

<Tabs>
<Tab title="Cách sử dụng">

```jsx
import { useBlockNote } from "@blocknote/react";
import { BlockEditor } from "@/ui/input/editor/components/BlockEditor";

export const MyComponent = () => {
  const BlockNoteEditor = useBlockNote();

  return <BlockEditor editor={BlockNoteEditor} />;
};
```

</Tab>
<Tab title="Props">


| Props | Type | Mô tả |
|-------|------|-------|
| editor | `BlockNoteEditor` | Instance hoặc cấu hình của block editor |
