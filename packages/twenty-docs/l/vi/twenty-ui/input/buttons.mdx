Danh sách các button và nhóm button được sử dụng xuyên suốt ứng dụng.

## Button

<Tabs>

<Tab title="Cách sử dụng">

```jsx
import { Button } from "@/ui/input/button/components/Button";

export const MyComponent = () => {
  return (
    <Button
      className
      Icon={null}
      title="Title"
      fullWidth={false}
      variant="primary"
      size="medium"
      position="standalone"
      accent="default"
      soon={false}
      disabled={false}
      focus={true}
      onClick={() => console.log("click")}
    />
  );
};
```
</Tab>

<Tab title="Props">


| Props | Type | Mô tả |
|-------|------|-------|
| className | string | Class tùy chọn để tùy chỉnh thêm kiểu dáng |
| Icon | `React.ComponentType` | Component biểu tượng tùy chọn được hiển thị bên trong button |
| title | string | Nội dung văn bản của button |
| fullWidth | boolean | Xác định xem button có nên chiếm toàn bộ chiều rộng của container hay không |
| variant | string | Biến thể kiểu dáng trực quan của button. Các tùy chọn bao gồm `primary`, `secondary`, và `tertiary` |
| size | string | Kích thước của button. Có hai tùy chọn: `small` và `medium` |
| position | string | Vị trí của button so với các phần tử đồng cấp. Các tùy chọn bao gồm: `standalone`, `left`, `right`, và `middle` |
| accent | string | Màu nhấn của button. Các tùy chọn bao gồm: `default`, `blue`, và `danger` |
| soon | boolean | Chỉ ra rằng button được đánh dấu là "sắp ra mắt" (dành cho các tính năng sắp tới) |
| disabled | boolean | Chỉ định xem button có bị vô hiệu hóa hay không |
| focus | boolean | Xác định xem button có đang được focus hay không |
| onClick | function | Hàm callback được kích hoạt khi người dùng nhấp vào button |




</Tab>

</Tabs>

## Button Group

<Tabs>
<Tab title="Cách sử dụng">
```jsx
import { Button } from "@/ui/input/button/components/Button";
import { ButtonGroup } from "@/ui/input/button/components/ButtonGroup";

export const MyComponent = () => {
  return (
    <ButtonGroup variant="primary" size="large" accent="blue" className>
      <Button
        className
        Icon={null}
        title="Button 1"
        fullWidth={false}
        variant="primary"
        size="medium"
        position="standalone"
        accent="blue"
        soon={false}
        disabled={false}
        focus={false}
        onClick={() => console.log("click")}
      />
      <Button
        className
        Icon={null}
        title="Button 2"
        fullWidth={false}
        variant="secondary"
        size="medium"
        position="left"
        accent="blue"
        soon={false}
        disabled={false}
        focus={false}
        onClick={() => console.log("click")}
      />
      <Button
        className
        Icon={null}
        title="Button 3"
        fullWidth={false}
        variant="tertiary"
        size="medium"
        position="right"
        accent="blue"
        soon={false}
        disabled={false}
        focus={false}
        onClick={() => console.log("click")}
      />
    </ButtonGroup>
  );
};

```
</Tab>

<Tab title="Props">


| Props | Type | Mô tả |
|-------|------|-------|
| variant | string | Biến thể kiểu dáng trực quan của các button trong nhóm. Các tùy chọn bao gồm `primary`, `secondary`, và `tertiary` |
| size | string | Kích thước của các button trong nhóm. Có hai tùy chọn: `medium` và `small` |
| accent | string | Màu nhấn của các button trong nhóm. Các tùy chọn bao gồm `default`, `blue` và `danger` |
| className | string | Class tùy chọn để tùy chỉnh thêm kiểu dáng |
| children | ReactNode | Một mảng các phần tử React đại diện cho các button riêng lẻ trong nhóm |



</Tab>
</Tabs>

## Floating Icon Button Group

<Tabs>
<Tab title="Cách sử dụng">

```jsx
import { FloatingIconButtonGroup } from "@/ui/input/button/components/FloatingIconButtonGroup";
import { IconClipboardText, IconCheckbox } from "@tabler/icons-react";

export const MyComponent = () => {
  const iconButtons = [
    {
      Icon: IconClipboardText,
      onClick: () => console.log("Button 1 clicked"),
      isActive: true,
    },
    {
      Icon: IconCheckbox,
      onClick: () => console.log("Button 2 clicked"),
      isActive: true,
    },
  ];

  return (
    <FloatingIconButtonGroup
    className
    size="small"
    iconButtons={iconButtons} />
  );
};

```

</Tab>

<Tab title="Props">


| Props | Type | Mô tả |
|-------|------|-------|
| className | string | Tên tùy chọn để tùy chỉnh thêm kiểu dáng |
| size | string | Kích thước của button. Có hai tùy chọn: `small` và `medium` |
| iconButtons | array | Một mảng các đối tượng, mỗi đối tượng đại diện cho một icon button trong nhóm. Mỗi đối tượng nên bao gồm component biểu tượng bạn muốn hiển thị trong button, hàm bạn muốn gọi khi người dùng nhấp vào button, và xem button có nên ở trạng thái active hay không. |



</Tab>

</Tabs>
