## Animated Checkmark

Biểu thị biểu tượng checkmark với tính năng hoạt ảnh bổ sung.

<Tabs>

<Tab title="Cách sử dụng">

```jsx
import { AnimatedCheckmark } from 'twenty-ui/display';

export const MyComponent = () => {
  return (
    <AnimatedCheckmark
      isAnimating={true}
      color="green"
      duration={0.5}
      size={30}
    />
  );
};
```

</Tab>

<Tab title="Props">


| Props | Type | Mô tả | Mặc định |
|-------|------|-------|----------|
| isAnimating | boolean | Kiểm soát xem checkmark có đang chạy hoạt ảnh hay không | false |
| color | string | Màu của checkmark |  |
| duration | number | Thời gian hoạt ảnh tính bằng giây | 0.5 giây |
| size | number | Kích thước của checkmark | 28 pixel |



</Tab>

</Tabs>
