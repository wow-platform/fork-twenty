Một mục menu đa năng được thiết kế để sử dụng trong menu hoặc danh sách điều hướng.

<Tabs>
<Tab title="Cách sử dụng">

```jsx
import { IconBell } from "@tabler/icons-react";
import { IconAlertCircle } from "@tabler/icons-react";
import { MenuItem } from "twenty-ui/display";

export const MyComponent = () => {
  const handleMenuItemClick = (event) => {
    console.log("Menu item clicked!", event);
  };

  const handleButtonClick = (event) => {
    console.log("Icon button clicked!", event);
  };

  return (
    <MenuItem
      LeftIcon={IconBell}
      accent="default"
      text="Menu item text"
      iconButtons={[{ Icon: IconAlertCircle, onClick: handleButtonClick }]}
      isTooltipOpen={true}
      testId="menu-item-1"
      onClick={handleMenuItemClick}
      className
    />
  );
};
```


</Tab>
<Tab title="Props">


| Props | Type | Mô tả |
|-------|------|-------|
| LeftIcon | IconComponent | Biểu tượng bên trái tùy chọn hiển thị trước văn bản trong mục menu |
| accent | string | Chỉ định màu nhấn của mục menu. Các tùy chọn bao gồm: `default`, `danger`, và `placeholder` |
| text | string | Nội dung văn bản của mục menu |
| iconButtons | array | Một mảng các đối tượng đại diện cho các icon button bổ sung liên kết với mục menu |
| isTooltipOpen | boolean | Kiểm soát khả năng hiển thị của tooltip liên kết với mục menu |
| testId | string | Thuộc tính data-testid cho mục đích kiểm thử |
| onClick | function | Hàm callback được kích hoạt khi mục menu được nhấp |
| className | string | Tên tùy chọn để tùy chỉnh thêm kiểu dáng |




</Tab>
</Tabs>


## Các biến thể

Các biến thể khác nhau của component menu item bao gồm:

### Command

Mục menu kiểu command trong menu để chỉ ra phím tắt.

<Tabs>
<Tab title="Cách sử dụng">

```jsx
import { IconBell } from "@tabler/icons-react";
import { MenuItemCommand } from "twenty-ui/display";

export const MyComponent = () => {
  const handleCommandClick = () => {
    console.log("Command clicked!");
  };

  return (
    <MenuItemCommand
      LeftIcon={IconBell}
      text="First Option"
      firstHotKey="⌘"
      secondHotKey="1"
      isSelected={true}
      onClick={handleCommandClick}
      className
    />
  );
};
```


</Tab>
<Tab title="Props">


| Props | Type | Mô tả |
|-------|------|-------|
| LeftIcon | IconComponent | Biểu tượng bên trái tùy chọn hiển thị trước văn bản trong mục menu |
| text | string | Nội dung văn bản của mục menu |
| firstHotKey | string | Phím tắt đầu tiên liên kết với command |
| secondHotKey | string | Phím tắt thứ hai liên kết với command |
| isSelected | boolean | Chỉ ra xem mục menu có đang được chọn hoặc được đánh dấu hay không |
| onClick | function | Hàm callback được kích hoạt khi mục menu được nhấp |
| className | string | Tên tùy chọn để tùy chỉnh thêm kiểu dáng |



</Tab>
</Tabs>

### Draggable

Component mục menu có thể kéo được thiết kế để sử dụng trong menu hoặc danh sách nơi các mục có thể được kéo, và các hành động bổ sung có thể được thực hiện thông qua các icon button.

<Tabs>
<Tab title="Cách sử dụng">

```jsx
import { IconBell } from "@tabler/icons-react";
import { IconAlertCircle } from "@tabler/icons-react";
import { MenuItemDraggable } from "twenty-ui/display";

export const MyComponent = () => {
  const handleMenuItemClick = (event) => {
    console.log("Menu item clicked!", event);
  };

  return (
    <MenuItemDraggable
      LeftIcon={IconBell}
      accent="default"
      iconButtons={[{ Icon: IconAlertCircle, onClick: handleButtonClick }]}
      isTooltipOpen={false}
      onClick={handleMenuItemClick}
      text="Menu item draggable"
      isDragDisabled={false}
      className
    />
  );
};
```

</Tab>
<Tab title="Props">


| Props | Type | Mô tả |
|-------|------|-------|
| LeftIcon | IconComponent | Biểu tượng bên trái tùy chọn hiển thị trước văn bản trong mục menu |
| accent | string | Màu nhấn của mục menu. Có thể là `default`, `placeholder`, và `danger` |
| iconButtons | array | Một mảng các đối tượng đại diện cho các icon button bổ sung liên kết với mục menu |
| isTooltipOpen | boolean | Kiểm soát khả năng hiển thị của tooltip liên kết với mục menu |
| onClick | function | Hàm callback được kích hoạt khi liên kết được nhấp |
| text | string | Nội dung văn bản của mục menu |
| isDragDisabled | boolean | Chỉ ra xem chức năng kéo có bị vô hiệu hóa hay không |
| className | string | Tên tùy chọn để tùy chỉnh thêm kiểu dáng |




</Tab>
</Tabs>

### Select Color

Mục menu có thể chọn với mẫu màu cho các trường hợp bạn muốn người dùng chọn một màu từ menu.

<Tabs>
<Tab title="Cách sử dụng">

```jsx
import { MenuItemSelectColor } from "twenty-ui/display";

export const MyComponent = () => {
  const handleSelection = () => {
    console.log("Menu item selected");
  };

  return (
    <MenuItemSelectColor
      color="green"
      selected={true}
      disabled={false}
      hovered={true}
      variant="default"
      onClick={handleSelection}
      className
    />
  );
};
```

</Tab>
<Tab title="Props">


| Props | Type | Mô tả |
|-------|------|-------|
| color | string | Màu chủ đề hiển thị dưới dạng mẫu trong mục menu. Các tùy chọn bao gồm: `green`, `turquoise`, `sky`, `blue`, `purple`, `pink`, `red`, `orange`, `yellow`, và `gray` |
| selected | boolean | Chỉ ra xem mục menu có đang được chọn (đánh dấu) hay không |
| disabled | boolean | Chỉ ra xem mục menu có bị vô hiệu hóa hay không |
| hovered | boolean | Chỉ ra xem mục menu có đang được di chuột qua hay không |
| variant | string | Biến thể của mẫu màu. Có thể là `default` hoặc `pipeline` |
| onClick | function | Hàm callback được kích hoạt khi mục menu được nhấp |
| className | string | Tên tùy chọn để tùy chỉnh thêm kiểu dáng |




</Tab>
</Tabs>

### Toggle

Mục menu với công tắc toggle đi kèm cho phép người dùng bật hoặc tắt một tính năng cụ thể.

<Tabs>
<Tab title="Cách sử dụng">

```jsx
import { IconBell } from '@tabler/icons-react';

import { MenuItemToggle } from 'twenty-ui/display';

export const MyComponent = () => {

  return (
    <MenuItemToggle
      LeftIcon={IconBell}
      text="First Option"
      toggled={true}
      toggleSize="small"
      className
    />
  );
};
```

</Tab>
<Tab title="Props">


| Props | Type | Mô tả |
|-------|------|-------|
| LeftIcon | IconComponent | Biểu tượng bên trái tùy chọn hiển thị trước văn bản trong mục menu |
| text | string | Nội dung văn bản của mục menu |
| toggled | boolean | Chỉ ra xem công tắc toggle đang ở trạng thái "bật" hay "tắt" |
| onToggleChange | function | Hàm callback được kích hoạt khi trạng thái công tắc toggle thay đổi |
| toggleSize | string | Kích thước của công tắc toggle. Có thể là `small` hoặc `medium` |
| className | string | Tên tùy chọn để tùy chỉnh thêm kiểu dáng |




</Tab>
</Tabs>
